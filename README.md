# Heart Disease Analysis & Clinical Dashboard

An end-to-end data visualization and analysis ecosystem designed to investigate risk factors behind heart disease clinical records. This repository hosts data assets alongside an interactive Power BI report built with clean, accessible corporate styling.

---

## 📊 Project Overview

Predicting cardiovascular events early is vital to clinical management. This project surfaces actionable health insights by highlighting acute physiological indicators against baseline statistics. 

### Core Components
* **Power BI Architecture:** Built with native layout mappings, embedded relational models, and dark/light accessible themes.
* **Pre-configured Visual Styles:** Custom parameters ensuring that text data, tooltips, and gridlines present clearly without visual clutter.
* **Optimized UI Assets:** Includes pre-cleaned canvas graphics and media resources for a seamless user experience.
